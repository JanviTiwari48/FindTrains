#  FindTrains

FindTrains is a Train Search web application that allows users to search for available trains between two stations using their station codes. The application is built with a Spring Boot backend, MySQL database, and a responsive HTML, CSS, and JavaScript frontend.

---

##  Features

- Search trains using source and destination station codes
- Display train name and train number
- Show departure and arrival timings
- Display source and destination station details
- Responsive and clean user interface
- REST API integration using Fetch API
- MySQL database for storing train schedules
